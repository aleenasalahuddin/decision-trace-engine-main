Decision Trace Engine is a web-based cognitive analysis tool designed to
deconstruct human decision-making into structured reasoning components.
It helps users surface hidden assumptions, cognitive biases, missing
information, and confidence misalignment behind important decisions.

## What it does
- Accepts a user-described decision
- Breaks the decision into explicit assumptions
- Identifies potential cognitive and emotional biases
- Highlights missing or unvalidated information
- Produces a structured decision trace report
- Generates a confidence and risk assessment summary

## Use cases
- Strategic and executive decision review
- Career and personal decision analysis
- Product and business tradeoff evaluation
- Cognitive bias awareness and reasoning audits

## Tech stack
- React + TypeScript
- Vite
- Tailwind CSS
- Component-driven UI architecture
